* 가이드 
언제나 이전 단계로 이동 가능 
부서(자기 참조 릴레시션 없음)과 직원 분류
일짜는 String 형식(예시. 2024-08-0)
한 달 출근일 24일
일부 데이터 사전 입력 필요(chatgpt 사용)
Option : 추가 제공 화면 자율


==== 인적 자원 관리 시스템 ====

1. 조직/직무 관리
2. 인사행정
3. 근태 관리 *
4. 급여/정산
5. 사회보험
6. 평가 관리
7. 연말정산
8. 승진 관리
9. 핵심 인재 관리
10. 월별 종합 현황 보기
0. 종료 *

선택하세요: 

==== 근태 관리 ====

1. 근태 입력 (option)
2. 근태 수정 *
3. 근태 삭제 *
4. 직원별 월별 근태 현황 보기 (option)
5. 부서별 월별 근태 현황 보기 *
0. 메인 메뉴로 돌아가기 *

선택하세요: 

==== 근태 입력 ====

직원 ID 입력: [EX: 12345]
날짜 입력 (YYYY-MM-DD): [EX: 2024-08-01]
근무 상태 입력 (출근/퇴근/휴가 등): [EX: 출근]

근태 정보가 입력되었습니다.
